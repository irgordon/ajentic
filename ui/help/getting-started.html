Getting started

Start the local UI from the UI folder with npm run dev.

The dev server prints a local browser URL. Open that URL in a browser on the same machine.

The first screen shows the AJENTIC local operator shell, the current harness status, the local transport boundary, workflow panels, and operator controls.

The broad workflow is simple. Set up the provider path, run constrained local steps, review validation, inspect a staged proposal, decide whether to continue, materialize local candidate output, review replay/status, export evidence, package the session, and use restore/history views when needed.

The help is explanatory. It is local-only and non-production.
