# Context pipeline

This document describes the structural role of the context pipeline in AJENTIC.

This document is subordinate to `docs/architecture/ARCHITECTURE.md`.

## Role

The context pipeline assembles bounded, inspectable context slices for model invocations.

Context packets must be auditable and carry provenance where required.

Context does not create authority by itself.
